# Required Documents

Applicants must submit the following documents during the loan application process:

## Identity Verification
- Government-issued photo ID
- Social Security Number

## Income Verification
- Recent pay stubs (last 30 days)
- W-2 forms (last 2 years)
- Tax returns for self-employed applicants

## Additional Documents
- Proof of address
- Bank statements (optional depending on loan type)
